# Internal Agent Notes

This directory contains two kinds of documents:

- Public docs: `roadmap.md`, `supported-devices.md` and user-facing guides.
- Internal build notes: `build-spec-*`, `HANDOFF.md`, `PROGRESS-*`,
  `APPLY-*`, `SUDO-*` and files under `docs/progress/`.

Internal build notes are kept while the project is under active multi-agent
development. They are useful context for future agents, but they are not
release documentation. Before publishing a public release, either move them to
a private planning area or scrub personal machine details from them.

Runtime code must not depend on internal docs.
