# ROG Monitor

Real-time ASUS ROG laptop monitoring tool for Linux.

## Features (V1)

* CPU average temperature
* CPU maximum temperature
* CPU minimum temperature
* Package temperature
* Count of cores above 90°C
* GPU temperature
* GPU utilization
* GPU power draw
* CPU, GPU and Mid fan RPM
* Estimated fan percentage
* Current power profile
* Current CPU governor
* Live terminal refresh using Rich

## Supported Hardware

Currently developed and tested on:

* ASUS ROG Strix G614JV
* Intel Core i7-13650HX
* NVIDIA RTX 4060 Laptop GPU
* Bazzite Linux
